電腦的發明讓許多簡單且需要重複的工作可以交給電腦做，而在 C 語言中有以下關鍵字來執行流程控制
```C
for, do-while, if-else, switch-case-default, goto, continue, break
```
## 1. 迴圈
迴圈是一種會重複執行某段敘述的流程控制，C 語言中的迴圈有 for 與 do-while，以下分別介紹
### for-loop
在 C 語言中，for 迴圈中有**起始條件; 結束條件; 變化條件**，但非必要。每個條件中也可以有不只一種條件。並用**大括號{}** 將想要重複的敘述包起來，此為 作用的**區域(scope)** ，那麼電腦就會重複執行大括號內的敘述直到達成結束條件。其結構如下
```C
// for(起始條件; 結束條件; 條件變化)
for(i = 0, j = 10; i < 10, j > 0; i = i+1, j = j-1)
{
    敘述1;
    敘述2;
    ...;
}
for(x = 0, y = 0; x+y < 10; x = x+1)
{
    敘述1;
    敘述2;
    ...;
}
```
在初學程式語言可能會對於 i = i+1 這敘述有疑惑，在數學上 = 是用來判斷兩邊是否相等，但是在程式語言中 = 是賦值的意思，在前面單純宣告變數可能還不明顯，但是在這邊的意思就很明顯，意思為 (新的)i 賦值為 (舊的)i+1，例如原本 i = 2(i 賦值 2)，那麼執行 i = i+1 後，新的 i 為 3。
若為巢狀迴圈，則會先執行內部迴圈在執行外部迴圈
```C
for(i = 0; i < 10; i = i+1)
{
    for(j = 0; j < 10; j = j+1)
    {
        敘述1;
        敘述2;
        ...;
    }
}
```
若是不會達到結束條件或沒有結束條件，則會造成無窮迴圈不會結束
```C
for(;;) //沒有結束條件
for(j = 0; j = 10; j = j-1) //不會達到結束條件
```
### do-while-loop
while 迴圈可以單獨使用，當條件達成後即停止此迴圈。其結構如下
```C
while(條件式)
{
    敘述1;
    敘述2;
    ...;
}
```
又因為在執行迴圈前會先判斷條件，故又稱**先判式迴圈**。也可以與 do 搭配，會先執行迴圈再判斷條件，故又稱**後判式迴圈**，其結構如下
```C
do
{
    敘述1;
    敘述2;
    ...;
}while(條件式);
```
**注意最後 while 要有分號**。
而在一般程式設計中，若清楚開始與結束條件時會使用 for-loop，若不清楚則使用 while-loop。而在部分情況中此兩個迴圈也可以交換使用，例如計算 1+2+3+...+10
```C
int j = 0, k = 0, i;
for(i = 0; i < 11; i++) j = j+i;

i = 0;
while(i < 11)
{
    k = k+i;
    i = i+1;
}
```
在迴圈中 i++ 可看成 i = i+1。若是迴圈內要執行的敘述只有一行，則可不用 {} ，但**建議都使用 {}** 可讓讀者更清楚。
## 2. 邏輯判斷
在邏輯判斷中，C 語言提供了 if-else, switch-case-default 兩個語法。使用邏輯判斷時會從上而下判斷是否有達成條件，若有則執行，無則跳下一個判斷式直到結束。
### if-else 判斷
if 可單獨使用，也可搭配 else 使用，也可組成 else if 使用。else if 與 else 非必須，若有則順序由上而下，且為 if... else if... else，且當其中一個條件達成後，其餘條件皆不執行。其結構如下
```C
if (條件1)
{
    敘述1-1;
    敘述1-2;
    ...;
}
else if (條件2)
{
    敘述2-1;
    敘述2-2;
    ...;
}
else if (條件3)
{
    敘述3-1;
    敘述3-2;
    ...;
}
else
{
    敘述4-1;
    敘述4-2;
    ...;
}
```
上述判斷時會先判斷條件 1 是否成立，若成立則會執行敘述 1-1, 1-2, ...，若不成立，則會往下判斷條件 2 ...，以此類推。若都不成立，則會進入 else 敘述執行。
其中條件的寫法很多樣，例如判斷一變數範圍可用 >(大於), <(小於), >=(大於或等於), <=(小於或等於)，判斷一變數是否為某值可用 == (等於)。判斷兩個以上的值可用 ||(或), &&(且)，例如
```C
if (a == 1 || a == 2)
{
    b = 2;
}
else if (a >= 3)
{
    b = 3;
}
else if (a < 0)
{
    b = 4;
}
else if (a == 4 && b > 6)
{
    b = 7;
}
else
{
    b = 10;
}
```
其語意為**若 a = 1 或 a = 2 時，b = 2。若 a >= 3 時，b = 3。若 a < 1 時，b = 4。若 a = 4 且 b > 6 時，b = 7。其餘條件，b = 10**。
若為巢狀判斷，會先判斷外面的條件是否成立，再往內判斷，順序由外而內，與 for-loop 相反。
```C
if (條件1)
{
    if(條件1-1)
    {
        敘述1-1;
        敘述1-2;
        ...;
    }
}
```
若為巢狀判斷，且有使用 else，在沒有 {} 區分的情況下，該 else 會與最近的 if 做搭配。例如
```C
if (條件1) 
    if(條件2) 
        敘述1; 
else 
    敘述2;
```
C 語言中使用 {} 來區分區段，其餘無嚴格限制，故在上述例子中的 else 會與條件 2 的 if 搭配，但是看起來會像式與條件 1 的 if 搭配。

### switch-case-default
switch 中會先放一個變數或運算式，進去後在把條件寫在 case 中。此與 if 最大的不同在於，switch 中有許多條件(case)，當條件 1 達成後會繼續判斷下一個條件是否達成，直到 switch 的區域結束為止。其結構如下
```C
switch(變數或運算式)
{
    case A:
        敘述1;
    case B:
        敘述2;
    default:
        敘述3;
}
```
要注意的是 case 後面是使用冒號而非分號。在上述例子中若是 A 的條件，則會執行敘述 1，不是則不執行。**但不論 A 的條件是否達成，都會判斷之後的條件**，最後到 default 的條件，會判斷是否滿足非 A 或非 B 的條件。因為後面沒有其他 case，故跳脫出 switch，default 的功能類似於 else。若是未使用 default，當 switch 中所有條件都未達成，則都不會執行。
若是有多個條件要使用同一個敘述，則可用以下寫法
```C
switch(變數或運算式)
{
    case A:
    case B:
        敘述1;
    case C:
    case D:
        敘述2;
    default:
        敘述3;
}
```
在上述例子中，若是達成 A 或 B 的條件，則執行敘述 1，並繼續往下判斷，若是達成 C 或 D 的條件，則執行敘述 2，並繼續到 default，判斷是否為非 A, B, C, D 的條件。
switch 與 if 最大的差別在於，if 只要當一個條件成立，其餘的 else if 與 else 都不會判斷，而 case 中的條件達成，依然會往下判斷。

## 跳脫流程
前面介紹了重複執行的敘述，以及達到某條件就執行某敘述，那麼若是有不想執行的敘述，則可使用下列語法。
### 1. break
break 可跳脫 for, while, do-while, switch 等流程，跳脫後執行下一個區域。例如
```C
int j = 0, k = 0, i;
for(i = 0; i < 11; i = i+1)
{
    j = j+i;
    if(i == 5) break;
}

i = 0;
while(i < 11)
{
    k = k+i;
    i = i+1;
    if(i == 5) break;
}
```
上述迴圈原本是計算 1+2+...+10，但是因為 if(i == 5) break，故變成 1+2+3+4+5。若是 switch 搭配 break 使用，其功能類似於 if。
```C
switch(x)
{
    case A:
        敘述1;
        break;
    case B:
        敘述2;
        break;
    default:
        敘述3;
}
```
上述例子中，若 x = A，則執行敘述 1 並跳脫 switch，若 x = B，則執行敘述 2 並跳脫 switch，若為其他條件，則執行敘述 3。
### 2. continue
不同於 break，此敘述為當達成某條件時**不執行此敘述之後的敘述**，例如要計算 1+2+3+4+6+7+8+9+10，可以寫兩個迴圈，也可以寫一個迴圈如下
```C
int j = 0, k = 0, i;
for(i = 0; i < 11; i = i+1)
{
    if(i == 5) continue; //不執行 +5
    //若此 if 在 j = j+i 後面，則會先執行 j+i，就沒達到想要的效果
    j = j+i; 
}
```

### 3. goto
goto 語法在 C 語言中較少見，除非團隊所有成員對於 C 語言非常了解，否則此用法會大大降低程式碼的可讀性以及維護困難。goto 在同一函數中往前或往後跳躍，其結構如下
```C
goto label;
label:
    敘述;
```
其中 label 並無限定必須出現在 goto 前或 goto 後，故會造成非開發者閱讀困難。但是 goto 也有許多好用的地方，例如一次跳出多層巢狀迴圈
```C
out_loop:
    sum = 10;
    goto end;
for(i = 0; i < 10; i = i+1)
{
    for(j = 0; j < 10; j = j+1)
    {
        for(k = 0; k < 10; k = k+1)
        {
            if(i == 9 && k == 5) goto out_loop;
        }
    }
}
end:
    return 0;
```
在上述例子中，程式會先進入巢狀迴圈，且當 i = 9 與 k = 5 時跳到 out_loop 標籤並跳出巢狀迴圈，並讓 sum = 10，最後在跳到 end 標籤回傳 0。此例顯示出 goto 的優點與缺點。優點是跳出巢狀迴圈很方便，缺點是程式碼會一直亂跳，讓讀者難以理解。故在許多非資深工程師寫的程式碼會非常少見。
